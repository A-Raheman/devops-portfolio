# Linux Basics for DevOps

## Objective

Learn essential Linux commands used in DevOps environments including file management, process management, and shell scripting.

---

## Topics Covered

- File and directory operations
- Permissions
- Process monitoring
- Bash scripting
- Cron jobs
- Service management

---

## Example Commands

List files
- ls -la

Change permissions
- chmod +x scripts.sh

Check running processes
- ps aux

---

## DevOps Importance

Linux powers most cloud servers and containers.  
Understanding Linux is critical for automation, debugging, and deployment.

---

## Lessons Learned

- Linux commands are fundamental for DevOps
- Bash scripting automates repetitive tasks
- System monitoring helps diagnose production issues
